\# Программа `single` — Hello / Goodbye world с IPC



\## Постановка задачи



Программа `single` должна:



\- выводить строку `Hello world` \*\*посимвольно\*\*, по одному символу в секунду;

\- если в системе уже есть \*\*другой процесс\*\*, одновременно исполняющий эту же программу и находящийся \*\*в стадии вывода\*\*, новый процесс должен выводить строку `Goodbye world`;

\- для определения факта «есть ли сейчас другой печатающий процесс» должны использоваться средства \*\*межпроцессного взаимодействия (IPC)\*\*;

\- необходимо учесть, что процесс может завершаться досрочно по `Ctrl-C` (`SIGINT`), и при этом общее состояние (IPC) не должно ломаться.



Формально: выбор строки (`Hello` или `Goodbye`) должен зависеть от того, сколько экземпляров `single` уже находятся в «стадии вывода» в момент старта нового процесса.
